# 5sim টেলিগ্রাম বট সেটআপ গাইড

এই প্রজেক্টটি 5sim.net API ব্যবহার করে অটোমেটিক ফোন নম্বর কেনা এবং টেলিগ্রাম অ্যাকাউন্ট চেক করার জন্য তৈরি করা হয়েছে।

## ১. লাইব্রেরি ইনস্টল করা
প্রথমে আপনার পিসিতে Python ইনস্টল করা থাকতে হবে। এরপর টার্মিনাল বা কমান্ড প্রম্পটে নিচের কমান্ডটি রান করুন:

```bash
pip install pyTelegramBotAPI requests telethon pycountry
```

## ২. কনফিগারেশন (main.py)
কোড পরিবর্তন না করে শুধুমাত্র উপরের দিকের এই ভেরিয়েবলগুলো আপনার তথ্য দিয়ে আপডেট করুন:

- `BOT_TOKEN`: আপনার বটের টোকেন।
- `ADMIN_ID`: আপনার টেলিগ্রাম আইডি।
- `API_ID` ও `API_HASH`: টেলিগ্রামের API তথ্য (my.telegram.org থেকে সংগৃহীত)।

## ৩. বট চালানো
টার্মিনালে নিচের কমান্ডটি লিখুন:

```bash
python main.py
```

## ৪. বট কানেক্ট করার নিয়ম
- টেলিগ্রামে বটটি ওপেন করে `/start` লিখুন।
- আপনার **5sim API Key** দিয়ে লগইন করুন।
- মেনু থেকে **"Setup Checker"** এ গিয়ে একটি টেলিগ্রাম অ্যাকাউন্ট লগইন করুন (এটি নম্বর চেক করার জন্য ব্যবহৃত হয়)।
- এরপর আপনি **"Buy Number"** অপশন থেকে নম্বর কিনতে পারবেন।

---
**সতর্কতা:** কোডের মূল লজিক পরিবর্তন করবেন না, শুধুমাত্র কনফিগারেশন অংশটি আপনার নিজের তথ্য দিয়ে পূরণ করুন।
